How to submit a project

For new ideas

  • Write at least a single sentence about how it helps people, and how it could work
  • Create a GitHub account, and submit your idea here
  • Set up a public repository somewhere and invite people who want to work on the project to it
  • Choose a license type and set this in your GitHub repository. (this is really important later on)
  • Once you have done everything above, ask a maintainer in the haxx.nz discord to create a project channel for you
  • Start making / jamming on design and code in that channel! Ask for help in the #design / #dev / #data channels if you need it.

For existing projects that you want to hand over maintainence of
